# 📝 贡献与反馈指南

感谢你关注本项目！为了确保你的反馈能得到及时处理，请遵循以下原则：

### 🚀 优先处理规则
- **详细表单优先：** 严格按照 Issue 模板填写、提供复现步骤和环境信息的 Issue 将会被**优先处理**。
- **拒绝模糊反馈：** 只有一句话描述、缺乏关键信息的 Issue，我们可能会直接关闭。

### 🛠 提交建议
1. 提交 Bug 前，请先确认是否为最新版本。
2. 尽可能提供截图或日志。
3. 如果是功能建议，请描述具体的应用场景。

### 📌 提交入口
- Bug 反馈：请使用仓库中的 `🐛 Bug 反馈` 模板。
- 功能建议：请使用仓库中的 `💡 功能建议` 模板。
